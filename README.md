# Nex-Trends Prospect Scraper
**Purpose:** Find ANY business with marketing, tech, or web service gaps.

## Features
- Multi-vertical gap detection (poor website, no social, tech complaints)
- Web presence analysis (site speed, mobile-friendly, SSL)
- Social media activity monitoring
- Tech stack detection
- Business size estimation
- 0-10 opportunity scoring

## Gap Types Detected
1. **Marketing Gaps:** No social presence, inactive accounts, poor branding
2. **Web Gaps:** Slow site, not mobile-friendly, outdated design, no SSL
3. **Tech Gaps:** No chat widget, no CRM, manual processes, no automation
4. **A2P Opportunities:** High call volume, missed calls, slow response times

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python src/main.py discover
```
